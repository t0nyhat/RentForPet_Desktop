# Обзор и возможности

## 1. Главная структура панели

В админ-панели доступны основные разделы:

- Обзор
- Расписание
- Все бронирования
- История платежей
- Создать вручную
- Типы номеров
- Номера
- Свободные номера
- Клиенты
- Настройки бронирования

![RU Placeholder: навигация админ-панели](/help/assets/ru/01-admin-navigation.png)

## 2. Бронирования

### 2.1 Расписание (Gantt + календарь)

- Просмотр бронирований по номерам и датам
- Изменение диапазона дат
- Режимы масштаба отображения
- Создание бронирования из расписания
- Объединение бронирований
- Обновление дат, назначение номера, перенос по номеру/датам

![RU Placeholder: экран расписания](/help/assets/ru/02-schedule.png)
![RU Placeholder: экран календаря](/help/assets/ru/02-calendar.png)

### 2.2 Журнал всех бронирований

- Таблица всех бронирований
- Поиск по клиентам/питомцам/номерам
- Быстрые фильтры по статусам
- Фильтрация по датам
- Сортировка и пагинация
- Операции по бронированию:
  - Подтвердить
  - Заселить
  - Выселить
  - Отменить
  - Удалить
  - Назначить номер
  - Изменить даты

![RU Placeholder: таблица бронирований](/help/assets/ru/03-bookings-table.png)

### 2.3 Ручное создание бронирования

- Выбор клиента и питомцев
- Выбор периода проживания
- Автоподбор вариантов размещения
- Поддержка составных вариантов (с переездами)
- Учёт скидки лояльности

![RU Placeholder: форма ручного бронирования](/help/assets/ru/04-manual-booking.png)

## 3. Финансы и оплаты

### 3.1 История платежей

- Полный список платежей
- Фильтры по статусу, способу, типу, датам
- Поиск по клиенту, бронированию, транзакции
- Просмотр деталей платежа

![RU Placeholder: история платежей](/help/assets/ru/05-payments-history.png)

### 3.2 Платёжные операции в бронировании

- Предоплата и полная оплата
- Изменение суммы предоплаты
- Отмена предоплаты
- Возврат средств
- Перенос переплаты на другое бронирование
- Конвертация переплаты
- Печать/просмотр чека

![RU Placeholder: операции оплаты в карточке бронирования](/help/assets/ru/06-booking-payments.png)

## 4. Клиенты и питомцы

- База клиентов с поиском
- Создание/редактирование/удаление клиента
- Скидка лояльности
- Карточки питомцев клиента
- Добавление/редактирование/удаление питомца
- Внутренние заметки по клиенту и питомцу

![RU Placeholder: раздел клиенты](/help/assets/ru/07-clients.png)

## 5. Номерной фонд

### 5.1 Типы номеров

- Создание и редактирование типов номеров
- Стоимость за единицу проживания
- Стоимость за доп. питомцев
- Вместимость, площадь, список удобств
- Активация/архивация типа

![RU Placeholder: типы номеров](/help/assets/ru/08-room-types.png)

### 5.2 Номера

- Добавление и редактирование реальных комнат
- Привязка к типу номера
- Фильтры и поиск
- Архивные/активные номера

![RU Placeholder: управление номерами](/help/assets/ru/09-rooms.png)

### 5.3 Свободные номера

- Подбор доступных номеров на заданный период
- Учет количества питомцев
- Группировка результатов по типу номера
- Предрасчет стоимости по выбранному периоду

![RU Placeholder: свободные номера](/help/assets/ru/10-available-rooms.png)
